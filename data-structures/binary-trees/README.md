# Binary Tree Implementation

A C++ program demonstrating binary tree operations using a binary tree abstract data type.

## Description

This project explores the implementation and use of a binary tree structure. The program creates binary trees, performs tree traversals, and retrieves information about the tree such as height and number of nodes.

## Features

- Create binary tree objects
- Add nodes to a tree
- Perform preorder traversal
- Perform inorder traversal
- Calculate tree height
- Count the number of nodes

## Concepts Practiced

- Binary trees
- Tree traversal algorithms
- Abstract Data Types (ADT)
- Interfaces
- Templates in C++
- Recursion concepts
- Object-oriented programming

## Language

C++
