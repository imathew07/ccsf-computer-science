# Dictionary Birthday Tracker

A C++ program that uses a Dictionary ADT to store and manage people's birthdays.

## Description

This project implements a birthday tracking system using a dictionary data structure.

Each person is stored using their name as the key and their birthday information as the associated value.

The program supports adding entries, searching for people, displaying birthdays by month, and removing entries.

## Features

- Add people and birthdays
- Search entries by name
- Remove entries
- Display all stored birthdays
- Filter people by birth month

## Data Structure Used

- Dictionary ADT

## Concepts Practiced

- Key-value storage
- Searching
- Insertion and deletion operations
- Abstract data types
- Object-oriented programming

## Language

C++
