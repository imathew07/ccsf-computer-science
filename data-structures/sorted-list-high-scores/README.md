# Sorted List High Scores

A C++ program that uses a sorted list data structure to manage and display high scores.

## Description

This project implements a high score system using a Sorted List Abstract Data Type.

Scores are automatically inserted in sorted order, allowing the program to maintain a ranking list while supporting insertion, retrieval, and list operations.

The project also tests various sorted list operations including insertion, removal, copying, and clearing the list.

## Features

- Add new scores
- Automatically maintain sorted order
- Display current rankings
- Remove entries
- Retrieve entries by position
- Handle invalid operations using exceptions

## Data Structure Implemented

Sorted List ADT

## Concepts Practiced

- Abstract Data Types (ADT)
- Lists
- Sorted insertion
- Templates
- Linked structures
- Exception handling
- Object-oriented design

## Language

C++
