# Graph Airline Routing

A C++ airline routing system using graph data structures and shortest path algorithms.

## Description

This project models an airline network using a graph represented by an adjacency matrix.

Cities are represented as vertices and flight connections are represented as weighted edges containing travel costs.

The program applies Dijkstra's shortest path algorithm to calculate the lowest cost routes from SFO to other destinations.

## Features

- Represent a network of cities using a graph
- Store connections using an adjacency matrix
- Calculate shortest paths using Dijkstra's algorithm
- Display graph representation
- Find lowest cost airline routes

## Data Structures and Algorithms

- Graph
- Adjacency Matrix
- Dijkstra's Shortest Path Algorithm
