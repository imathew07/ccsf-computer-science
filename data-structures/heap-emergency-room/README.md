# Heap Emergency Room

A C++ emergency room simulation using a heap-based priority queue.

## Description

This project implements an emergency room management system where patients are organized according to their priority level.

A heap data structure is used to efficiently insert patients and retrieve the highest priority patient.

The Patient class overloads comparison operators to allow the heap to organize patients based on priority.

## Features

- Create patient objects
- Assign priority levels
- Compare patients using overloaded operators
- Manage patients using a heap priority queue
- Simulate emergency room scheduling

## Data Structure Implemented

Heap / Priority Queue
