# Infix Postfix Calculator

A C++ calculator that converts mathematical expressions from infix notation to postfix notation and evaluates the result using a stack.

## Description

This project implements an expression calculator using a stack data structure.

The program accepts mathematical expressions written in infix notation, converts them into postfix notation using operator precedence rules, and evaluates the postfix expression.

## Features

- Convert infix expressions to postfix notation
- Evaluate postfix expressions
- Handle operator precedence
- Support parentheses
- Detect invalid expressions
- Detect unmatched parentheses
- Detect division by zero
- Validate operands using stack operations

## Data Structures Used

- Stack

## Algorithms and Concepts

- Infix to postfix conversion
- Expression parsing
- Operator precedence
- Stack-based evaluation

## Language

C++
