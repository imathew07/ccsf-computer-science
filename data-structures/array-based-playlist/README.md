# Array-Based Playlist

A C++ implementation of a dynamic array-based list used to manage a music playlist.

## Description

This project implements a custom List data structure using a dynamically allocated array.

The list supports inserting, removing, modifying, and retrieving items while automatically resizing the internal array when more space is required.

A playlist application is built using this list structure to demonstrate how the data structure works in practice.

## Features

- Implement a custom List class
- Insert songs at specific positions
- Remove songs from the playlist
- Modify existing entries
- Retrieve songs by position
- Dynamically resize the underlying array
- Handle invalid positions using exceptions

## Data Structure Implemented

Array-Based List
