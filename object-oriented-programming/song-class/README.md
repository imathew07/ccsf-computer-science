# Song Class

A C++ program demonstrating basic object-oriented programming concepts using a Song class.

## Description

This project creates a Song class that stores information about a song, including its name and artist.

The program uses private attributes with setter and getter methods to demonstrate encapsulation and object interaction.

## Features

- Create Song objects
- Store song name and artist information
- Modify object data using setter methods
- Retrieve object data using getter methods
- Display song information

## Concepts Practiced

- Classes and objects
- Encapsulation
- Private attributes
- Getter and setter methods
- Object-oriented programming

## Language

C++
