# Recursive Functions

A collection of C++ recursive function exercises demonstrating different recursion techniques.

## Description

This project explores recursion through multiple examples, including reversing numbers, reversing strings, and calculating sums using recursive function calls.

The exercises demonstrate how base cases and recursive calls work together to solve problems.

## Features

- Reverse digits of an integer recursively
- Reverse strings using different recursive approaches
- Calculate the sum of a range of integers recursively
- Demonstrate recursive call execution order

## Concepts Practiced

- Recursion
- Base cases
- Recursive calls
- Call stack behavior
- String manipulation
- Problem decomposition

## Language

C++
