# Template Box Search

A C++ program demonstrating templates, classes, and exception handling.

## Description

This project implements a generic Box class using C++ templates.

The program stores integer values inside Box objects and searches through an array of boxes to find a target value.

The project also demonstrates exception handling when invalid input is provided or when a value cannot be found.

## Features

- Generic class using C++ templates
- Store and retrieve data using class methods
- Search through an array of objects
- Handle errors using exceptions

## Concepts Practiced

- Object-oriented programming
- C++ templates
- Classes and objects
- Encapsulation
- Exception handling
- Linear search

## Language

C++
